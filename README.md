# Projeto Delivery de Cafeteria

## Ambiente de Desenvolvimento
- Windows 10
- VSCode
- Python 3.12

## Tecnologias Utilizadas
- Linguagem: Python 3.12

## Como Instalar e Executar

```bash
git clone https://github.com/seuusuario/seurepositorio.git
cd seurepositorio
python -m venv .venv
